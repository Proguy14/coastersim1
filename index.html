Physics breakdown

The three investigations

INVESTIGATION 01
Brake pad material
Tests how rubber, felt, cardboard, and sandpaper affect stopping distance via friction coefficient μ.
INVESTIGATION 02
Cart mass
Tests how added mass (0g, 200g, 400g) affects momentum p=mv and therefore stopping distance.
INVESTIGATION 03
Release height
Tests how ramp height controls entry speed via KE=mgh → v=√(2gh) and its effect on braking distance.
